总结：
.bowerrc文件是自定义bower下载的代码包的目录

bower init 会生成 bower.json

bower install jquery --save bower下载最新版本到配置的目录下，自动修改bower.json下的dependencies下的依赖

bower info jquery  会看到可用的版本信息

bower update 切换版本

bower search bootstrap 列出包涵bootsrtap的可用包


bower uninstall jquery  卸载jquery

bower list 列出安装的应用程序包


bower help 查看相关命令
